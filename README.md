# Ultimate Jingles for Cocoon

The Ultimate Video Game Jingle Repo for Cocoon.

This repository currently contains **3,123 jingles** across Nintendo, Sega, PlayStation, Xbox, Steam, Android, PC Engine, and several smaller niche platforms.

What started as a large Nintendo-focused collection has grown into a broad multi-platform jingle library for Cocoon, with enough depth to cover both common games and more niche libraries.

## Adding the repo to Cocoon

Add this repository through `Cocoon Settings -> Jingle Settings -> Add Repository`.

After that, you can individually choose which jingle you want per game.

## Matching

The repo includes a proper `index.json` with regex matching, so Cocoon has a much better chance of finding the right jingle even when game names are written differently.

## Goal

The goal is to make this the biggest and most useful general jingle repo for Cocoon.

Instead of being a small single-platform pack, this repo aims to give broad coverage across handhelds, consoles, PC releases, mobile games, and niche systems while also keeping match quality high.

## Contributions

If you know missing jingles, find wrong matches, notice missing games, or have cleaner versions of existing jingles, feel free to reach out or open a contribution.

The aim is for this repo to be a go-to jingle library for Cocoon instead of another half-finished collection.
